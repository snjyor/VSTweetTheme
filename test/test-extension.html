🎨 VS Tweet Theme 扩展测试

📋 测试说明: 这个页面帮助你验证VS Tweet Theme扩展是否正确安装和配置。请按顺序完成以下测试项目。

🔧 1. 扩展安装检查

  • 扩展已在Chrome中加载 待检查
  • 扩展图标出现在工具栏 待检查
  • 扩展状态为"已启用" 待检查
  • 没有错误提示信息 待检查

检查方法:

  1. 访问 chrome://extensions/
  2. 找到"VS Tweet Theme"扩展
  3. 确认开关处于开启状态
  4. 检查是否有错误信息

🌐 2. Twitter页面测试

  • 能够访问Twitter/X.com 待测试

  • 弹出面板正常显示 待测试

测试步骤:

  1. 打开新标签页访问 Twitter.com 或 X.com
  2. 点击工具栏中的VS Tweet Theme图标
  3. 检查弹出面板是否正常显示
  4. 观察状态指示器颜色和文本

🎨 3. VSCode模式测试

  • 能够启用VSCode模式 待测试

  • 推文显示为代码注释 待测试
  • 媒体内容被正确处理 待测试
  • 能够正常切换回原始界面 待测试

测试步骤:

  1. 在Twitter页面点击"启用VSCode模式"按钮
  2. 观察页面是否转换为深色主题
  3. 检查推文是否显示为代码注释格式
  4. 查看图片/视频是否被隐藏或显示为占位符
  5. 再次点击按钮确认能够恢复原始界面
