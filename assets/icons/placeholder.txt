# VS Tweet Theme 图标文件

## 当前状态
已创建 icon.svg 作为基础图标设计，包含：
- VSCode窗口样式
- Twitter鸟图标元素
- 代码符号装饰
- VSCode经典配色方案

## 需要的PNG文件
请将以下尺寸的图标文件放置在此目录中：

- icon16.png  (16x16像素)
- icon32.png  (32x32像素)  
- icon48.png  (48x48像素)
- icon128.png (128x128像素)

## 生成PNG文件的方法

### 方法一：在线转换
1. 使用 icon.svg 作为基础
2. 访问 https://svgtopng.com/ 或类似工具
3. 上传 icon.svg 并设置对应尺寸
4. 下载生成的PNG文件

### 方法二：使用设计工具
- Figma: 导入SVG，导出为PNG
- Adobe Illustrator: 打开SVG，导出为PNG
- Inkscape: 免费工具，支持SVG到PNG转换

### 方法三：命令行工具
```bash
# 使用 ImageMagick
convert icon.svg -resize 16x16 icon16.png
convert icon.svg -resize 32x32 icon32.png
convert icon.svg -resize 48x48 icon48.png
convert icon.svg -resize 128x128 icon128.png
```

## 图标设计说明
- 主色调：VSCode蓝色 (#007ACC)
- 背景：深色主题 (#1E1E1E)
